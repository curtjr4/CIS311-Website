College Skills

While I was attending Arapahoe Community College and Colorado State University at Pueblo, I have learned a number of valuable skills that pertain to Computer Information Systems as well as Cybersecurity. Many of the notable hard skills that I have learned include:

  • Vulnerability Assessment
  • Basic Linux System Administration
  • Firewalls and how they work (not vendor specific)
  • Basic Cisco iOS Syntax
  • Basic Microsoft Windows Server Administration
  • Hypervisors/System Virtualization (VMWare ESXi)
  • Computer Networking (Trained with CompTIA Network+ Material)
  • Website Creation (HTML, CSS)
  • Exposure the Java, Python, and C# Programming Languages
  • Network Scanning Tools (Greenbone, Nessus, NMAP)
  • Basic Cybersecurity Concepts (Trained with CompTIA Security+ Material)
  • Metasploit
  • Lockheed Martin Cyber Killchain
  • Internet Privacy/Evasion Techniques

Although the degree program proved to be rigorous and exposed us to many materials, the above is some of the most notable and is not an accurate reflection of my entire college career, as well as the skills that I have learned while working an IT job where I get exposure to many other products and technologies, such as SD-WAN, microcegmentation, as well as gaining experience with hyperconvered infrastructure such as the Nutanix platform and the Acropolis Hypervisor.
